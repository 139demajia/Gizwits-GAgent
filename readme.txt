
----------------------------------------------- GAgent 运行须知 -----------------------------------------------

1.运行之前检查串口设备号，然后根据实际更改串口名称到文件 platform.h 中的 UART_NAME 以及指定网卡设备 NET_ADAPTHER
2.必须把 gagent2-0/arch/linux/dll 文件夹下面的libGAgent.so，添加到你的系统环境变量里面。
3.运行GAgent_linux 必须用root权限。
